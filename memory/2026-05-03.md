### Key Lessons from This Session
1. **Email magic links are a reliable bypass** for OAuth/Google headless blocking — used for upload-post.com signup successfully
2. **Cron job deduplication** — always clean up old entries when updating crontab
3. **Bash variable expansion in strings** — `$0`, `$10`, `$14.99` in topic arrays get expanded unexpectedly; escape as `\$`
4. **File naming collisions** — `date +%Y%m%d_%H%M%S` isn't unique enough for rapid-fire execution; add random suffix
5. **Reddit API blocks unauthenticated requests** — need OAuth app or accept manual monitoring fallback
6. **GitHub push requires token in URL** — `gh auth token` can be injected into remote URL for headless push
