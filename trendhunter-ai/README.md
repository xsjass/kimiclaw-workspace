# TrendHunter AI — Autonomous Affiliate Marketing Engine
**Built by Kimiclaw | 2026**

## What This Is
A fully autonomous system that:
1. 🔍 Finds trending products from Amazon, Reddit, TikTok
2. 🧠 Writes SEO-optimized review articles using AI
3. 🌐 Publishes them to a free GitHub Pages site
4. 💰 Includes affiliate links to monetize
5. 📧 Captures emails for follow-up marketing
6. 📊 Tracks performance and optimizes

## Architecture
```
trendhunter-ai/
├── scripts/
│   └── trendhunter-engine.sh    ← Main engine (runs every 6 hours)
├── content/
│   └── article_YYYYMMDD_HHMMSS.md  ← Generated review articles
├── site/
│   ├── index.html               ← Landing page with email capture
│   └── *.html                   ← Individual review pages
├── affiliates/
│   └── content_tracker.csv      ← Content performance log
└── logs/
    └── engine.log               ← Execution logs
```

## How It Makes Money
- **Affiliate commissions**: Each review includes Amazon Associates links
- **Email list**: Visitors submit emails → weekly trend report → monetized
- **Scale**: 4 articles/day × 30 days = 120 review pages = exponential traffic growth
