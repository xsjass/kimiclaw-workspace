1. **GEMINI_API_KEY** → https://aistudio.google.com/app/apikey (free, no credit card)

### Usage
```bash
# Setup (run once)
./scripts/viraloop-setup.sh

# Edit credentials
nano ~/.openclaw/workspace/.env.viraloop

# Run manually
source ~/.openclaw/workspace/.env.viraloop
./scripts/viraloop-master.sh https://beatsheaven.com curiosity

# Schedule daily (add to crontab)
0 9 * * * cd /root/.openclaw/workspace && ./scripts/viraloop-daily.sh
```
